# DeepL Spider

```bash
git clone https://github.com/eadomenech/deepl_spider.git
```

```bash
cd deepl_spider/
```

Create a file called **.env** with the following content:

```bash
CLIENT_USER=user
PASSWORD=password
```

Create env:

```bash
python3 -m venv venv
```

```bash
source venv/bin/activate
```

```bash
python -m pip install -U pip
```

Install requeriments:

```bash
pip install -r requirements.txt
```

Run spider:

```bash
python main.py
```
